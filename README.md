# Сайт-портфолио
---
## **Выбранная тема: MkDocs Material**
```yml
theme:
  name: material
  language: ru
  palette:
    - scheme: default      # Светлая тема
    - scheme: slate        # Тёмная тема
  primary: deep purple
  accent: pink
```
---
## 🔍 Почему именно эта тема?
| Критерий | Обоснование|
|----------|------------|
| Документация | MkDocs Material имеет отличную документацию на русском и английском, что упрощает разработку и поддержку (то есть удобно) |
| Тёмная/светлая тема | Переключатель позволяет пользователю выбрать комфортный режим просмотра |
| Подсветка кода | Встроенная поддержка ``content.code.copy`` упрощает чтение и копирование примеров кода из лабораторных работ |
| Локализация | Полная поддержка русского языка (``language: ru``) — интерфейсные элементы отображаются корректно |
---
## 🎨 Почему выбраны цвета Deep Purple + Pink?
```yml
primary: deep purple
accent: pink
```
| Аспект | Обоснование|
|----------|------------|
| Читаемость | Высокий контраст между текстом и фоном |
| Современность | Цветовая палитра соответствует Material Design 3, что делает сайт визуально актуальным |
| Личное предпочтение | Фиолетовая тематика, по моему мнению, выглядит красиво |
| Акцентный цвет | Розовый акцент выделяет интерактивные элементы (ссылки, кнопки), не перегружая интерфейс |

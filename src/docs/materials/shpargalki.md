## Быстрые команды

```bash
# Запустить локальный сервер
mkdocs serve 

# Запустить локальный сервер с функцией редактирования в реальном времени 
mkdocs serve --livereload

# Собрать сайт
mkdocs build

# Обновить зависимости
pip install --upgrade mkdocs mkdocs-material
```

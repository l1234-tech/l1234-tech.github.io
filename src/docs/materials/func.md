---
title: ✨ Функции темы
---

```yaml
features:
  - navigation.tabs
  - navigation.tabs.sticky
  - navigation.expand
  - search.highlight
  - content.code.copy
```

---

| Функция | Описание | Зачем нужно |
|--------|--------|----------|
| ``navigation.tabs`` | Горизонтальные вкладки | Удобная навигация по разделам |
| ``navigation.tabs.sticky`` | Вкладки фиксируются при скролле | Доступ к меню всегда виден |
| ``navigation.expand`` | Авто-раскрытие подразделов | 	Не нужно кликать для просмотра |
| ``search.highlight`` | Подсветка текста в поиске | Быстрый поиск по контенту |
| ``content.code.copy`` | Кнопка копирования кода | Упрощает копирование примеров |

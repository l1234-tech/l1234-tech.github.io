## 🔍 Как пользоваться отчётами

1.  Откройте нужную лабораторную работу
2.  Изучите **цель** и **задание**
3.  Просмотрите **исходный код** (кнопка :material-content-copy: для копирования)
4.  Проверьте **результаты выполнения** (скриншоты, вывод консоли)
5.  Ознакомьтесь с **выводами**
